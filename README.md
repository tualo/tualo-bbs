# BBS Tools

This package is a collection for BBS frankin machines.


## Debuging

The following list of enviroment variables can be used the get extended debug informations

  * DEBUG_BBS_STARTJOB
  * DEBUG_BBS_STOPJOB
  * DEBUG_BBS_STATUS
  * DEBUG_BBS_MSG
  * DEBUG_BBS_SEQUENCE
  * DEBUG_BBS_IMPRINT
  * DEBUG_BBS_CONTROLLER
  * DEBUG_BBS_HTTPSERVER
